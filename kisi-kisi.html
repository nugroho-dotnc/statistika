Distribusi Frekuensi

Bab 2 — Deskriptif Statistik

⭐ Pengertian Statistika (Kisi Essay)

Statistika adalah ilmu yang mempelajari cara mengumpulkan, mengolah, menyajikan, menganalisis data, membuat kesimpulan, dan mengambil keputusan berdasarkan hasil analisis data tersebut.

Statistika Deskriptif adalah bagian statistika yang hanya mengolah dan menyajikan data tanpa menarik kesimpulan untuk populasi yang lebih besar. Cakupannya meliputi: distribusi frekuensi, ukuran pemusatan, ukuran dispersi, dan grafik.

Statistik = data/informasi itu sendiri (hasil penerapan algoritma pada data).
Statistika = ilmu yang mempelajari metode pengolahan data.

Tiga hal penting dalam statistika: (1) Data historis (nilai numerik masa lampau), (2) Kriteria keputusan (berbagai pilihan dengan manfaat & konsekuensi), (3) Pengambilan keputusan (penerapan nyata di dunia pendidikan, bisnis, dll).

Distribusi Frekuensi Berkelompok

Dibuat ketika data cukup banyak sehingga penyajian biasa tidak efisien. Data dikelompokkan ke dalam kelas-kelas interval beserta frekuensinya.

Rumus Sturges — menentukan jumlah kelas (K)
K = 1 + 3,3 · log(n)
n = jumlah data. Contoh: n=150 → K = 1 + 3,3 × log(150) = 1 + 3,3 × 2,18 ≈ 8 atau 9
Panjang Kelas (interval kelas)
I = Rentang / K = (data_max − data_min + 1) / K
1
Tentukan jumlah kelas (K) menggunakan rumus Sturges atau pengalaman (6–15 kelas).
2
Hitung rentang = data terbesar − data terkecil + 1.
3
Hitung panjang kelas = Rentang ÷ K. Bulatkan ke atas bila perlu.
4
Susun interval kelas mulai dari nilai terkecil (bisa dibulatkan agar komunikatif).
5
Masukkan data dengan tally — coret setiap data yang sudah masuk ke kelasnya.
Contoh — Nilai 150 Mahasiswa

n=150, data min=13, max=93 → K = 1+3,3×log(150) ≈ 9, Rentang = 93−13+1 = 81, Panjang Kelas = 81/9 = 9 (digunakan 10 agar komunikatif, mulai dari 10)

Ukuran Pemusatan Data

Mean (Rata-rata)

Data Tunggal
x̄ = ΣX / n
Data Berkelompok — Sigma
x̄ = Σ(fᵢ · xᵢ) / n
xᵢ = nilai tengah kelas
Data Berkelompok — Coding
x̄ = x₀ + (Σfᵢcᵢ / n) · I
x₀ = titik tengah kelas dg frekuensi terbesar, cᵢ = kode (-2,-1,0,1,2,...)
Data Berkelompok — Rata-rata Duga
x̄ = x₀ + Σ(fᵢ · dᵢ) / n
dᵢ = xᵢ − x₀

Modus (Mo)

Data Berkelompok
Mo = L₀ + C · [f₁/(f₁+f₂)]
L₀ = batas bawah kelas modus, C = interval kelas, f₁ = selisih frek. modus dg sebelumnya, f₂ = selisih frek. modus dg sesudahnya

Median (Me)

Data Berkelompok
Me = L₀ + C · [(n/2 − Σf₁)/fₘ]
L₀ = batas bawah kelas median, Σf₁ = jumlah frek. kelas sebelumnya, fₘ = frek. kelas median
Contoh — Modus Data Berkelompok

Data: 60–62 (f=4), 63–65 (f=10), 66–68 (f=17), 69–71 (f=9), 72–74 (f=5)

f₁ = 17−10=7, f₂ = 17−9=8, L₀ = 65,5, C = 3

Mo = 65,5 + 3×[7/(7+8)] = 65,5 + 1,4 = 66,9

Peluang (Probabilitas)

Bab 3 — Kisi Essay: Peluang & Probabilitas

⭐ Konsep Dasar Peluang (Kisi Essay)

Peluang adalah ukuran kemungkinan terjadinya suatu kejadian, dinyatakan dengan bilangan real p di mana 0 ≤ P(A) ≤ 1.

Ruang Sampel (S): himpunan semua kemungkinan hasil dari suatu percobaan.
Contoh: lempar koin → S = {A, G}
Kejadian (Event K): himpunan bagian dari ruang sampel.
Contoh: muncul bilangan genap dari dadu → K = {2, 4, 6}
Peluang Suatu Kejadian (Ruang Sampel Seragam)
P(K) = n(K) / n(S)
n(K) = banyak hasil yang memenuhi kejadian K, n(S) = banyak semua hasil yang mungkin
Sifat Peluang:
(1) 0 ≤ P(A) ≤ 1 untuk setiap kejadian A
(2) P(S) = 1 (jumlah seluruh peluang = 1)
(3) P(∅) = 0 (peluang kejadian mustahil = 0)
Contoh

Dua koin dilempar. Ruang sampel: S = {(G,G),(G,A),(A,G),(A,A)}, n(S)=4

Kejadian K = muncul tepat satu gambar = {(G,A),(A,G)}, n(K)=2

P(K) = 2/4 = 0,5

Aturan Peluang

Aturan Penjumlahan (Umum)
P(A∪B) = P(A)+P(B)−P(A∩B)
Jika A dan B saling lepas (mutually exclusive):
P(A∪B) = P(A) + P(B)
Aturan Perkalian (Umum)
P(A∩B) = P(A) · P(B|A)
Jika A dan B saling bebas (independent):
P(A∩B) = P(A) · P(B)
Peluang Bersyarat
P(A|B) = P(A∩B) / P(B), P(B)≠0
Peluang terjadinya A, diketahui B sudah terjadi
Contoh — Peluang Bersyarat

Perusahaan Highfly: P(pesanan siap dikirim tepat waktu R) = 0,80

P(pesanan siap DAN sampai tepat waktu, R∩D) = 0,72

P(sampai tepat waktu | sudah siap) = P(R∩D)/P(R) = 0,72/0,80 = 0,90

Permutasi & Kombinasi

Permutasi (urutan diperhatikan)

P(n,r) — r dari n unsur berbeda
P(n,r) = n! / (n−r)!
Semua n unsur berbeda
Pₙ = n!
Ada unsur yang sama (k kelompok)
P = n! / (n₁! · n₂! · ... · nₖ!)

Kombinasi (urutan tidak diperhatikan)

C(n,r) — memilih r dari n
C(n,r) = n! / [r! · (n−r)!]
0! = 1
Perbedaan utama:
Permutasi: AB ≠ BA (urutan beda = beda susunan)
Kombinasi: AB = BA (urutan tidak dipedulikan)
Contoh — Kombinasi

5 bab buku, ingin baca 3 bab saja. Berapa cara?

C(5,3) = 5! / (3! × 2!) = 120 / (6×2) = 10 cara

Contoh — Permutasi

Dari 8 RW, 2 RW dipilih untuk bertanding (urutan penting: kandang/tandang). Berapa cara?

P(8,2) = 8! / (8−2)! = 8×7 = 56 cara

Distribusi Normal

Bab 5 — Kisi Essay: 2 Soal (Tabel Z)

⭐ Pengertian & Ciri-ciri

Distribusi Normal (Gaussian) adalah distribusi probabilitas yang paling banyak digunakan dalam statistika. Grafik fungsi kepadatan probabilitasnya berbentuk kurva lonceng (bell curve).

Ciri-ciri Distribusi Normal:
a. Kurva berbentuk garis lengkung halus seperti genta/lonceng
b. Simetris terhadap rata-rata (mean) μ
c. Kedua ekor mendekati sumbu-x tapi tidak pernah menyentuh (asimtot)
d. Jarak titik belok dari sumbu simetris = σ (simpangan baku)
e. Luas total di bawah kurva = 1 (100%)
Fungsi Kepadatan Probabilitas Normal X ~ N(μ, σ²)
f(x; μ, σ) = (1 / σ√2π) · e^[−(x−μ)² / 2σ²]
μ = rata-rata, σ = simpangan baku, π ≈ 3.14159, e ≈ 2.71828
Aturan Empiris (68-95-99.7):
• μ ± 1σ → sekitar 68% data
• μ ± 2σ → sekitar 95% data
• μ ± 3σ → sekitar 99,7% data
Parameter:
• Notasi: X ~ N(μ, σ²)
• Mean = Median = Modus = μ
• Varian = σ²

⭐ Distribusi Normal Baku (Z) & Tabel Z

Karena integral distribusi normal tidak bisa diselesaikan secara analitis, digunakan Distribusi Normal Baku Z ~ N(0, 1) dengan mean=0 dan σ=1.

Transformasi ke Z (Standardisasi)
Z = (X − μ) / σ
Nilai Z dari tabel adalah P(Z < z₀) = luas area dari −∞ sampai z₀
⚠️ Ingat saat baca tabel Z: Tabel memberikan P(Z < z₀) — yaitu luas di sebelah KIRI dari z₀. Untuk P(Z > z₀) = 1 − P(Z < z₀).
Yang DicariRumusContoh
P(Z < z₀)Langsung baca tabelP(Z < 1.84) = 0,9671
P(Z > z₀)1 − P(Z < z₀)P(Z > 1.84) = 1 − 0,9671 = 0,0329
P(a < Z < b)P(Z < b) − P(Z < a)P(−1.97 < Z < 0.86) = 0,8051 − 0,0244 = 0,7807
P(Z < −z₀)1 − P(Z < z₀)P(Z < −1.25) = 1 − P(Z < 1.25) = 1 − 0,8944 = 0,1056
Contoh Lengkap — Tabel Z (dari buku)

Umur bola lampu terdistribusi normal, μ = 800 jam, σ = 40 jam.

(a) P(778 < X < 834) = ?

z₁ = (778−800)/40 = −0,55 → P(Z < −0,55) = 0,2912

z₂ = (834−800)/40 = 0,85 → P(Z < 0,85) = 0,8023

P(778 < X < 834) = 0,8023 − 0,2912 = 0,5111

(b) P(X < 750 atau X > 900) = ?

z₁ = (750−800)/40 = −1,25 → P(Z < −1,25) = 0,1056

z₂ = (900−800)/40 = 2,5 → P(Z > 2,5) = 1 − 0,9938 = 0,0062

P = 0,1056 + 0,0062 = 0,1118

Contoh — Mencari Nilai z

(a) P(Z > k) = 0,3015 → P(Z < k) = 1 − 0,3015 = 0,6985 → dari tabel: k = 0,52

(b) P(k < Z < −0,18) = 0,4197 → P(Z < −0,18) = 0,4286

0,4286 − P(Z < k) = 0,4197 → P(Z < k) = 0,0089 → dari tabel: k = −2,37

Aproksimasi Normal terhadap Distribusi Binomial

Bila n cukup besar dan np ≥ 5 serta nq ≥ 5 (q = 1−p), distribusi Binomial dapat didekati oleh distribusi Normal.

Rumus Aproksimasi
Z = (X − np) / √(npq)
Gunakan koreksi kontinyuitas ±0,5 untuk keakuratan lebih baik

Distribusi Gamma

Distribusi Kontinu — Kisi Essay: 1 Soal (Tabel Gamma)

⭐ Fungsi Gamma & Distribusi Gamma

Distribusi Gamma adalah distribusi peluang kontinu yang digunakan untuk memodelkan waktu tunggu hingga suatu kejadian terjadi sebanyak α kali, atau data yang menjulur ke kanan (positively skewed). Ini adalah generalisasi dari distribusi Eksponensial.

Fungsi Gamma — Γ(α)
Γ(α) = ∫₀^∞ x^(α−1) · e^(−x) dx, α > 0
Sifat penting: Γ(α) = (α−1) · Γ(α−1) dan Γ(n) = (n−1)! untuk n bilangan bulat positif
Sifat Khusus Fungsi Gamma:
• Γ(1) = 1
• Γ(1/2) = √π
• Γ(n) = (n−1)! jika n ∈ ℤ⁺
• Γ(α) = (α−1)·Γ(α−1)
Contoh:
• Γ(4) = 3! = 6
• Γ(5) = 4! = 24
• Γ(3) = 2! = 2
• Γ(3/2) = (1/2)·Γ(1/2) = √π/2

Fungsi Kepadatan Probabilitas Gamma

PDF Distribusi Gamma X ~ Gamma(α, β)
f(x; α, β) = [1 / (βᵅ · Γ(α))] · x^(α−1) · e^(−x/β), x > 0
α = parameter bentuk (shape), β = parameter skala (scale)
Nilai Ekspektasi & Varians
E(X) = α·β Var(X) = α·β²
ParameterSimbolKeterangan
Shape (bentuk)α (alpha)Mengontrol bentuk distribusi. α=1 → eksponensial
Scale (skala)β (beta)Mengontrol skala/penyebaran distribusi
Meanμ = αβNilai rata-rata distribusi
Variansσ² = αβ²Ukuran penyebaran distribusi
Kasus Khusus Distribusi Gamma:
• Jika α = 1 → Distribusi Eksponensial dengan parameter β
• Jika α = n/2 dan β = 2 → Distribusi Chi-Square dengan df = n
• Jika α = bilangan bulat positif → Distribusi Erlang

Penggunaan Tabel Gamma & Cara Hitung Probabilitas

Untuk menghitung P(X ≤ x) pada distribusi Gamma, dilakukan transformasi ke variabel standar:

Transformasi Standar
Y = X / β → Y ~ Gamma(α, 1)
Lalu gunakan tabel distribusi Gamma kumulatif dengan parameter α
Fungsi Gamma Tidak Lengkap (untuk tabel)
P(X ≤ x) = γ(α, x/β) / Γ(α)
γ(α, u) = ∫₀ᵘ t^(α−1)·e^(−t) dt adalah fungsi gamma tidak lengkap
Contoh — Distribusi Gamma

Waktu pelayanan pelanggan X ~ Gamma(α=2, β=3 menit).

Hitung: rata-rata, varians, dan P(X ≤ 6).

E(X) = α·β = 2 × 3 = 6 menit

Var(X) = α·β² = 2 × 9 = 18 menit²

Untuk P(X ≤ 6): transformasi Y = X/β = X/3, sehingga P(Y ≤ 2) dengan Y ~ Gamma(2,1)

Dari tabel Gamma atau hitung: P(Y ≤ 2) = 1 − e⁻² (1 + 2) = 1 − 3e⁻² ≈ 1 − 0,406 = 0,594

Contoh — Menghitung E(X) dan Var(X)

X ~ Gamma(α=3, β=4). Cari E(X), Var(X), dan P(X > 0).

E(X) = α·β = 3 × 4 = 12

Var(X) = α·β² = 3 × 16 = 48

P(X > 0) = 1 (karena distribusi Gamma hanya terdefinisi untuk x > 0)

Tips Tabel Gamma untuk UTS:
Tabel Gamma biasanya menampilkan P(X ≤ x) atau nilai fungsi Gamma Γ(α). Pastikan:
1. Tentukan parameter α dan β dari soal
2. Transformasi x → y = x/β sebelum melihat tabel
3. Hitung Γ(n) = (n−1)! untuk n bilangan bulat
4. Gunakan sifat Γ(α) = (α−1)·Γ(α−1) jika α bukan bilangan bulat

Kunci Jawaban:

Statistika adalah ilmu yang mempelajari cara mengumpulkan, mengolah, menyajikan, menganalisis data, membuat kesimpulan, dan mengambil keputusan berdasarkan hasil analisis tersebut.

Statistik adalah data itu sendiri, informasi, atau hasil penerapan algoritma statistika pada suatu data.

Perbedaan: Statistika = ilmunya; Statistik = datanya/hasilnya.

Tiga hal penting dalam Statistika:

1. Data historis — nilai numerik dari masa lampau yang diolah menjadi informasi untuk pengambilan keputusan di masa depan.

2. Kriteria keputusan — berbagai pilihan yang memiliki manfaat dan konsekuensi. Statistika membantu menentukan kriteria terbaik dari pilihan-pilihan tersebut.

3. Pengambilan keputusan — peran utama statistika adalah membantu pengambilan keputusan yang tepat, contoh: penentuan penerimaan siswa berdasarkan nilai UNAS.

Peluang / Probabilitas
Sebuah kotak berisi 5 bola merah, 3 bola biru, dan 2 bola hijau. Diambil 1 bola secara acak.

a) Tentukan ruang sampel dan banyaknya anggota ruang sampel n(S)!
b) Berapa peluang terambilnya bola merah?
c) Berapa peluang terambilnya bola bukan hijau?
d) Jika diambil 2 bola secara berurutan tanpa pengembalian, berapa peluang bola pertama merah DAN bola kedua biru?
Kunci Jawaban:

a) n(S) = 5+3+2 = 10 bola. Ruang sampel S = {M₁, M₂, M₃, M₄, M₅, B₁, B₂, B₃, H₁, H₂}

b) K = {bola merah}, n(K) = 5
P(merah) = n(K)/n(S) = 5/10 = 1/2 = 0,5

c) Bukan hijau = merah + biru = 5+3 = 8 bola
P(bukan hijau) = 8/10 = 4/5 = 0,8

d) Tanpa pengembalian → gunakan aturan perkalian (peluang bersyarat):
P(pertama merah) = 5/10 = 1/2
P(kedua biru | pertama merah) = 3/9 = 1/3 (tersisa 9 bola)
P(merah, lalu biru) = (5/10) × (3/9) = 15/90 = 1/6 ≈ 0,167

Distribusi Normal — Tabel Z (1)
Nilai ujian mahasiswa Teknik Informatika terdistribusi normal dengan rata-rata μ = 72 dan simpangan baku σ = 8.

a) Hitunglah P(X < 80)!
b) Hitunglah P(X > 60)!
c) Hitunglah P(64 < X < 88)!
d) Berapa nilai minimum seorang mahasiswa agar termasuk 10% terbaik (P(X > x) = 0,10)?

Gunakan: P(Z < 1,00)=0,8413; P(Z < 1,50)=0,9332; P(Z < 2,00)=0,9772; P(Z < 1,28)=0,8997; P(Z < −1,50)=0,0668; P(Z < −1,00)=0,1587
Kunci Jawaban:

a) P(X < 80):
Z = (80−72)/8 = 8/8 = 1,00
P(X < 80) = P(Z < 1,00) = 0,8413

b) P(X > 60):
Z = (60−72)/8 = −12/8 = −1,50
P(X > 60) = 1 − P(Z < −1,50) = 1 − 0,0668 = 0,9332

c) P(64 < X < 88):
z₁ = (64−72)/8 = −1,00 → P(Z < −1,00) = 0,1587
z₂ = (88−72)/8 = 2,00 → P(Z < 2,00) = 0,9772
P(64 < X < 88) = 0,9772 − 0,1587 = 0,8185

d) 10% terbaik → P(X > x) = 0,10 → P(X < x) = 0,90:
Dari tabel, P(Z < 1,28) = 0,8997 ≈ 0,90, jadi z ≈ 1,28
x = μ + z·σ = 72 + 1,28×8 = 72 + 10,24 = 82,24
Jadi nilai minimum ≈ 82,24 (dibulatkan 83)

Distribusi Normal — Tabel Z (2)
Berat badan mahasiswa PNJ terdistribusi normal dengan μ = 60 kg dan σ = 10 kg.

a) Berapa persen mahasiswa yang beratnya antara 50 kg dan 75 kg?
b) Berapa persen mahasiswa yang beratnya lebih dari 80 kg?
c) Jika terdapat 500 mahasiswa, berapa orang yang beratnya kurang dari 45 kg?

Gunakan: P(Z<−1,00)=0,1587; P(Z<−1,50)=0,0668; P(Z<1,50)=0,9332; P(Z<2,00)=0,9772; P(Z<−2,00)=0,0228
Kunci Jawaban:

a) P(50 < X < 75):
z₁ = (50−60)/10 = −1,00 → P(Z < −1,00) = 0,1587
z₂ = (75−60)/10 = 1,50 → P(Z < 1,50) = 0,9332
P(50 < X < 75) = 0,9332 − 0,1587 = 0,7745 = 77,45%

b) P(X > 80):
Z = (80−60)/10 = 2,00
P(X > 80) = 1 − P(Z < 2,00) = 1 − 0,9772 = 0,0228 = 2,28%

c) P(X < 45) × 500 orang:
Z = (45−60)/10 = −1,50
P(X < 45) = P(Z < −1,50) = 0,0668
Jumlah mahasiswa = 0,0668 × 500 = 33,4 ≈ 33 orang

Distribusi Gamma — Tabel Gamma
Diketahui X adalah variabel acak yang mengikuti distribusi Gamma dengan parameter α = 3 dan β = 2.

a) Tuliskan fungsi kepadatan probabilitas (PDF) dari X!
b) Hitung nilai E(X) dan Var(X)!
c) Hitung nilai Γ(4) dan Γ(7) menggunakan sifat fungsi Gamma!
d) Jika Y ~ Gamma(α=2, β=1), hitung E(Y²) menggunakan rumus E(Y²) = Var(Y) + [E(Y)]²!
Kunci Jawaban:

a) PDF X ~ Gamma(α=3, β=2):
f(x) = [1/(2³·Γ(3))] · x^(3−1) · e^(−x/2)
Γ(3) = 2! = 2; 2³ = 8
f(x) = [1/(8·2)] · x² · e^(−x/2) = x²·e^(−x/2)/16, untuk x > 0

b) E(X) dan Var(X):
E(X) = α·β = 3 × 2 = 6
Var(X) = α·β² = 3 × 4 = 12

c) Γ(4) dan Γ(7):
Γ(4) = (4−1)! = 3! = 6
Γ(7) = (7−1)! = 6! = 6×5×4×3×2×1 = 720
Atau rekursif: Γ(7) = 6·Γ(6) = 6·5·Γ(5) = 6·5·4·Γ(4) = 6·5·4·6 = 720 ✓

d) Y ~ Gamma(α=2, β=1):
E(Y) = α·β = 2·1 = 2
Var(Y) = α·β² = 2·1 = 2
E(Y²) = Var(Y) + [E(Y)]² = 2 + 2² = 2 + 4 = 6
